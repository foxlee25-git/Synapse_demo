Settings & Governance

让企业知道谁能看什么、谁能做什么、AI 能动到哪里为止。

这页把角色权限、连接授权、模型路由、Token 预算和 Agent 能访问的数据范围集中管理。

Roles

角色权限

  • 老板：看结果、批关键动作
  • 运营：看任务、改知识、管审批
  • 销售：看线索、跟进建议、客户摘要
  • 管理员：管连接、模型、预算和审计

Agent Scope

Agent 可访问范围

  • Growth Agent 仅访问广告和增长数据
  • Sales Agent 可访问 CRM 和邮件摘要
  • Knowledge Agent 可访问 SOP 与 FAQ 文档

Model Routing

模型策略

  • 摘要和分类默认 SLM
  • 业务复盘与建议用 Hybrid
  • 高价值推理或审批说明用 LLM
