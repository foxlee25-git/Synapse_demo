First-run Setup

Connect Agents, tools and permission boundaries before the dashboard opens.

正式产品的第一步不是聊天，而是建立组织状态网络：谁的 Agent 接入、能读哪些系统、能写回哪里、哪些动作必须审批。

1Choose workspace role

CEO、CTO、PM、Employee 进入不同默认视图，权限策略自动绑定。

2Connect employee Agents

MCP / Skill / API 接入 15 个员工 Agent，同步 presence、任务、阻塞和产物。

3Authorize SaaS sources

Slack、GitHub、Linear、Docs、CRM 以只读优先接入，写回动作走审批。

4Apply governance policy

按角色、团队、项目、敏感度定义可见字段和自动化边界。
